第三章 顺序结构

1.数据的表现

①常量（运行过程中，值不可改变。包括常数和代表固定不变的名称）：整型（3）、实型（4.5）、字符（'A'）、字符串（''ABC''）、布尔（Flase/True）
    常量除了常数，还可用符号常量：#define  <符号常量名>  <字符序列>
                             #define  PI  3.1415926

②变量（运行过程中，值可改变）：变量名必须为合法标志符。规则：由字母、数字和下划线'_'组成，且以字母或下划线开头。
    PS：使用前必须定义

③常变量： const  <类型名>  <常变量名>  =  <处值表达式>
         const  int a = 3;
    PS:符号常量和常变量区别
    符号常量用一个字符串代替一个符号，在预编译时将所有符号常量替换为所指定字符串，没有类型，在内存中并不存在以符号常量命名的存储单元。
    常变量有变量特征，有类型，在内存中存在着以他命名的存储单元，可以用sizeof运算符计算长度，与一般变量不同在于在运行期间不可改变。
